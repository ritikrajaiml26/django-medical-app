# Django Medical Reminder App

## Project Description
This is a Django based web application that helps users manage their medicine schedules and get reminders.

## Features
- User Registration & Login
- Add / Edit / Delete Medicine
- Reminder Dashboard
- Profile Management
- Responsive UI

## Tech Stack
- Python
- Django
- SQLite
- HTML / CSS
- Gunicorn
- Render Deployment

## Installation (Local Setup)
```bash
git clone https://github.com/yourusername/django-medical-app.git
cd django-medical-app
pip install -r requirements.txt
python manage.py runserver
